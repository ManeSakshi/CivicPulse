# Data Directory

This directory contains datasets used for the CivicPulse sentiment analysis project.

## Structure

- `external/` - External datasets (not tracked in Git due to size)
- `raw/` - Raw data collected from APIs
- `processed/` - Processed and cleaned data

## Required External Datasets

### 1. Sentiment140 Dataset

- **File**: `external/Sentiment140.csv` (227+ MB)
- **Source**: [Sentiment140 Dataset](http://help.sentiment140.com/for-students/)
- **Description**: 1.6 million tweets with sentiment labels
- **Download**: Extract to `data/external/Sentiment140.csv`

### 2. Additional Tweet Dataset

- **File**: `external/Tweets.csv` (~3.4 MB)
- **Description**: Additional tweet data for training
- **Place in**: `data/external/Tweets.csv`

## Setup Instructions

1. Download the Sentiment140 dataset from the official source
2. Extract and place as `data/external/Sentiment140.csv`
3. Place any additional datasets in the `external/` folder
4. Run the data collection scripts to populate `raw/` folder

## Generated Files

The following files are generated by the data collection scripts:

### News Data (Generated by `fetch_news_unified.py`)

- `raw/all_news_data.csv` - **Main consolidated news** from all sources
- `raw/news_data.csv` - NewsAPI articles only (for compatibility)
- `raw/gnews_data.csv` - GNews articles only (for compatibility)
- `raw/local_news.csv` - **Sangli-specific local news** (English + Marathi)

### Social Media Data

- `raw/twitter_data.csv` - Tweets from Twitter API/scraping (via `fetch_twitter_hybrid.py`)

### Legacy Note

Files are maintained in separate formats for backward compatibility with existing processing scripts.

These files are tracked in Git with LFS for smaller sizes.
